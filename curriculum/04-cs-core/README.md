# 04 — CS Core Subjects

In interviews — especially **TCS Technical, Infosys Technical, Wipro Technical, Cognizant Technical** — the interviewer almost always picks **2-3 questions from each of these 4 subjects**:

1. **DBMS** — Database Management Systems.
2. **OS** — Operating Systems.
3. **CN** — Computer Networks.
4. **OOPs** — Object-Oriented Programming Concepts.

You don't need to read 500-page textbooks. You need crisp, **interview-grade** answers for the ~80 most-asked questions. That's what this module gives you.

## Files

- [`dbms.md`](dbms.md)
- [`operating-systems.md`](operating-systems.md)
- [`computer-networks.md`](computer-networks.md)
- [`oop.md`](oop.md)

Each file follows the same structure:
1. **Concept tree** — the topics, in order.
2. **Must-know terms** — quick definitions.
3. **Interview Q&A** — the questions actually asked, with model 2-3 sentence answers.
4. **Diagrams you should be able to draw** — small ASCII / mental diagrams.
5. **Quick revision card** — what to re-read the night before an interview.

## Study plan

- One subject in **Phase 1-2** (DBMS), one in **Phase 2-3** (OS), one in **Phase 3** (CN), and **OOPs in parallel** with Python OOP work in Week 5-6.
- 30-45 min daily.
- Re-do **all 4 quick revision cards** every Sunday in 15 minutes.

## Why these four (and not others)

These are the only subjects almost guaranteed to come up. Subjects like **Compiler Design, Software Engineering, Theory of Computation** appear *occasionally* (1-2 questions), so save them for last and only if you have time.
