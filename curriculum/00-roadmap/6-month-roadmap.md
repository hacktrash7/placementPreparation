# The 6-Month Roadmap (Bird's-Eye View)

> Detailed week-by-week breakdown lives in [`../08-schedule/week-by-week.md`](../08-schedule/week-by-week.md). This file is the *summary card* — keep it printed/pinned.

The plan is split into **4 phases × ~6 weeks each**. Every phase has **one main coding theme, one CS subject, and ongoing aptitude/English**.

```
Phase 1 (Weeks  1- 6) : Foundations
Phase 2 (Weeks  7-12) : Core DSA + DBMS/OS
Phase 3 (Weeks 13-18) : Advanced DSA + CN/OOPs + One project
Phase 4 (Weeks 19-24) : Company-specific prep + Mocks + Resume/HR
                ──────────────────────────
After Week 24      : Maintenance mode — daily mocks until placement.
```

## Phase 1 — Foundations (Weeks 1-6)

**Goal:** Stop being scared of code. Build the daily habit.

- **Programming:** Python from zero → comfortable with loops, lists, dicts, strings, functions, files. (See `02-programming/`.)
- **DSA (light):** Big-O intuition, arrays, strings — solve 30 easy problems.
- **Aptitude:** Percentages, Ratios, Averages, Profit/Loss, Time-Speed-Distance basics.
- **English:** Reading comprehension daily + 5 new words/day.
- **CS subject:** DBMS basics — what is a database, SQL `SELECT/WHERE/JOIN`.
- **Bonus:** Set up GitHub, learn `git add/commit/push`, write your first README.
- **Output by Week 6:**
  - 30+ Python problems solved on HackerRank / LeetCode Easy.
  - Score 60%+ on a TCS NQT Quant mini-mock.
  - 1 tiny project on GitHub (e.g., command-line calculator or to-do list).

## Phase 2 — Core DSA + Theory (Weeks 7-12)

**Goal:** Reach a level where you can solve 1 easy + 1 medium DSA problem in 45 min.

- **DSA:** Recursion, Linked Lists, Stacks, Queues, Hash Maps, Sorting, Searching. ~100 problems total cumulative.
- **Aptitude:** Permutation & Combination, Probability, Number System, Logical Reasoning (blood relations, coding-decoding, syllogisms).
- **English:** Versant-style speaking practice 10 min/day. Essay writing weekly.
- **CS subject:** Finish **DBMS** (normalization, transactions, indexing) + start **OS** (processes, threads, scheduling, deadlocks).
- **Bonus:** Solid SQL — 50 problems on LeetCode / HackerRank SQL track.
- **Output by Week 12:**
  - Solve 1 medium DSA in 25 min consistently.
  - Score 70%+ on a full TCS NQT mock.
  - Resume v1 ready (1 page).

## Phase 3 — Advanced DSA + Project (Weeks 13-18)

**Goal:** Lock in interview-grade DSA and finish a project worth talking about.

- **DSA:** Trees, BST, Heaps, Graphs (BFS/DFS), basic DP. ~150 problems cumulative.
- **Aptitude:** Data Interpretation, advanced puzzles, mixtures, time & work, geometry.
- **English:** Group Discussion practice (record yourself), telephone-style speaking.
- **CS subject:** Finish **OS**, complete **Computer Networks** (OSI, TCP/IP, HTTP, DNS), **OOPs** (4 pillars + Java/Python examples).
- **Project:** Build **one full-stack project** OR **one data project** worth a paragraph in the resume. (See `06-resume-projects/project-ideas.md`.)
- **Bonus:** AWS Cloud Practitioner concepts (free tier playground) + AI/ML literacy module.
- **Output by Week 18:**
  - 150+ DSA problems solved.
  - 1 well-documented GitHub project deployed somewhere (Streamlit / Vercel / GitHub Pages).
  - Score 80%+ on a full Infosys-pattern mock.

## Phase 4 — Company-Specific Prep + Mocks (Weeks 19-24)

**Goal:** Drill, polish, and convert.

- **Mocks:** 1 full-length mock every Sunday (TCS / Infosys / Wipro / Cognizant in rotation). Review wrong answers carefully.
- **DSA:** Company-tagged problem lists (Striver SDE sheet — first 50, GFG company-wise top 50 for TCS/Infy).
- **Aptitude:** Sectional speed drills — 20 Qs in 20 min.
- **CS theory:** Daily 20-question viva drill from `04-cs-core/`.
- **HR & soft skills:** Tell-me-about-yourself, "Why TCS?", strengths/weaknesses, situational answers. Mock HR with a friend on weekends.
- **Resume v2:** Final version. Get it reviewed by a senior who has been placed.
- **Output by Week 24:**
  - Comfortable scoring 80%+ on any pattern test.
  - Crisp 2-minute self-intro.
  - Active GitHub, LinkedIn, polished resume PDF, mock HR done at least 5×.

## After Week 24 — Maintenance & Final Drive Phase

- 1 hour aptitude + 2 DSA problems + 1 mock HR question **every day**.
- Apply to every company that visits campus + off-campus drives (Superset, AMCAT, eLitmus).
- Keep a **rejection log**: what failed and what you'll fix before the next test.
