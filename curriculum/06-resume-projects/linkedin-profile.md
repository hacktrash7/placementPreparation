# LinkedIn Profile Playbook for Freshers

A polished LinkedIn profile = (a) recruiters find you, (b) referrals come easier, (c) you can stalk where seniors got placed.

Spend **one Saturday** doing this end-to-end.

## Profile checklist

- [ ] **Profile photo** — clear face, neutral background, smile. Phone selfie in good light works.
- [ ] **Banner image** — something neutral (city skyline, lines, abstract). Free at Canva.
- [ ] **Headline** (the line below your name):
      `B.Tech CSE '26 @ XYZ College | Python • SQL • Flask | Open to Software Engineering Internships`
- [ ] **About section** — 4-6 lines, first person, plain English.
- [ ] **Education** — Add B.Tech with start/end years, grade.
- [ ] **Experience** — Internships, even small ones. Add 2-3 lines of bullets each.
- [ ] **Projects** — Yes, LinkedIn has a Projects section. Mirror your resume's top 2.
- [ ] **Skills** — top 10. Hit "Add skill" and select all that apply. Get classmates to endorse 1-2.
- [ ] **Licenses & certifications** — HackerRank, Coursera, AWS, NPTEL.
- [ ] **Volunteer / Positions of Responsibility**.
- [ ] **Contact info** — Add personal email, GitHub URL, portfolio URL.

## Sample "About" section

```
B.Tech CSE undergraduate at XYZ College (2026 batch). I write Python
day-to-day and enjoy building small things that solve real problems —
recently shipped 'TaskFlow', a multi-user to-do web app with Flask + SQLite,
and 'IPL Insights', an interactive Streamlit dashboard on 16 seasons of
IPL data.

Currently going deep on DSA (180+ problems on LeetCode), SQL, and AWS
fundamentals while preparing for full-time software engineering roles.

Open to internships and full-time roles in Software Engineering and
Data Analytics. Reach me at ananya.sharma@gmail.com.
```

## Headline formulae that work

```
<Year> <Branch> @ <College> | <Top 3 skills> | Open to <role>
```

Examples:

- `B.Tech CSE '26 @ XYZ | Python · SQL · ML | Open to SDE Internships`
- `Final-year CSE | Full-stack Flask Developer | AWS CCP (in progress)`

Avoid: "Aspiring Software Engineer 💻", "Tech Enthusiast 🚀", "Programmer | Coder | Developer". They sound exactly like every other student.

## Using LinkedIn for real placement value

### 1. Find seniors who got placed at TCS / Infosys / Wipro

Search `XYZ College of Engineering` → filter by Current Company → message politely:

```
Hi [Name], I'm Ananya, a final-year CSE student at XYZ College.
I saw you joined TCS Digital last year — congratulations! If you
have 10 minutes, I'd love to ask you a few questions about your
prep approach. No pressure if you're busy. Thanks!
```

3 out of 10 will respond. That's gold.

### 2. Follow company pages

TCS Careers, Infosys Careers, Wipro Careers, Cognizant Careers, etc. Job alerts and drive announcements show up first.

### 3. Engage thoughtfully (don't spam)

Post 1-2 things a month:
- "Just finished my IPL Insights dashboard — here's what I learned about Pandas pivot tables."
- "Reflections on hitting 100 LeetCode solved."
- "Wrote a blog post on the 3-way TCP handshake — useful before any networks interview."

These small, genuine posts make your profile look *active* and recruiters notice.

### 4. Use LinkedIn job filters

- Filter: India + Entry-level + Software Engineer + Past 7 days.
- Save searches; LinkedIn emails you weekly.

### 5. Easy Apply only if your resume is ready

Don't apply with a half-baked resume. Get yourself ready first.

## Common mistakes

- **Spelling errors.** Recruiters notice.
- **Adding 500 random connections quickly.** Quality > quantity. Add classmates, seniors, recruiters who post relevant roles.
- **Cringe headlines** ("Synergistic Solutions Architect Wannabe"). No.
- **Buying followers / certificate sellers.** Will get you banned.
- **Posting daily memes.** This is a professional network.

## A 60-minute LinkedIn refresh plan

```
0-10  min : Update photo + banner.
10-25 min : Rewrite headline + about section.
25-40 min : Add 2 projects with descriptions + links.
40-50 min : Add skills + request 5 endorsements from classmates.
50-60 min : Connect with 20 placed seniors from your college.
```

After this, **revisit monthly** for 15 minutes. That's all the LinkedIn work you'll need.
