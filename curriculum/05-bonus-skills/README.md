# 05 — Bonus / In-Demand Skills (Resume Differentiators)

Service MNCs hire mostly on aptitude + coding + CS theory, but **two or three modern skills on your resume** will:

1. Get you shortlisted for **better-package tracks** (TCS Digital, Infosys DSE, Cognizant Elevate).
2. Give you talking points in HR ("I built a small data dashboard using Python + SQL").
3. Set you apart in **off-campus drives** and small product startups.

You only need **2-3 skills picked from below** to put on your resume — not all of them. Choose based on interest.

## How to pick

Most balanced choice for a service-MNC candidate from average college:

1. **SQL** (mandatory — no exceptions).
2. **Git/GitHub + Linux basics** (mandatory).
3. Pick **one** of (Web Dev OR AI/ML OR Cloud OR Data Analytics) based on interest.

That's it. Three line-items in your resume's "Skills" section, plus **one project** demonstrating them, will already make you stand out at this college tier.
