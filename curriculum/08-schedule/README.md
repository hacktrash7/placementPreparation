# 08 — The Schedule

A roadmap is useless without execution. This folder gives you the **exact day-by-day plan** to run the curriculum.

## Files

- [`week-by-week.md`](week-by-week.md) — 24-week breakdown with weekly goals and topic assignments.
- [`daily-template.md`](daily-template.md) — A reusable daily-log template (copy into Notion / paper).
- [`mock-test-cadence.md`](mock-test-cadence.md) — When and how to take full-length mocks.

## How to use

1. **Print or pin** the week-by-week table at your desk.
2. Each morning, open the daily template, fill it in *before* you start.
3. Each Sunday, do a 20-minute review: what worked, what didn't, what to adjust.
4. Adjust dates as needed — the *order* matters more than the calendar.

## A note on flexibility

Real life happens — illness, exams, family. Build in slack: every 4 weeks should have **at least 1 buffer day** that's not pre-scheduled, so you can absorb a missed day without panic.

But: **don't reschedule more than 3 times in a row.** Catch up that same weekend.
