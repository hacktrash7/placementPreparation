# Placement Preparation Curriculum

A complete, self-paced placement-preparation roadmap for a **3rd-year B.Tech CSE student** from an average-tier private college, targeting **service-based MNCs** (TCS, Infosys, Tech Mahindra, Wipro, Cognizant, Capgemini, Accenture, LTIMindtree, etc.) and ready to add **modern in-demand skills** to the resume.

> **Starting assumption:** the learner knows *very basic* coding (loops, if/else, can write a small program with help) and is in the second half of 3rd year. Placement season for these companies typically starts in the **July-November window of the final year**, so this plan is built for **~6 months of serious, consistent work** before that, followed by ongoing revision.

---

## Who this is for

- 3rd-year B.Tech / B.E. student, CSE / IT / ECE branch.
- Currently weak in coding, aptitude, and English fluency.
- Wants to crack mass-recruiter MNCs first, and use the same prep as a base for slightly better product/service hybrid companies (TCS Digital, Infosys Power Programmer, Wipro Elite, etc.).
- Studying **mostly alone**, on a laptop, with limited budget (everything here can be done with free or very cheap resources).

## What you'll be able to do at the end

1. Clear **TCS NQT, Infosys SP/DSE, Tech Mahindra, Wipro Elite NLTH, Cognizant GenC, Accenture, Capgemini** style assessments.
2. Solve **easy + most medium** DSA problems in Python (or Java/C++) within time pressure.
3. Answer core CS theory (DBMS, OS, CN, OOPs) confidently for technical interviews.
4. Write **clean SQL**, use **Git/GitHub**, and demo a small **cloud / web / data project**.
5. Carry a **1-page resume + 2 polished projects + a clean GitHub** into interviews.
6. Handle HR / managerial rounds with structured answers.

## How to use this repo

Work through the folders **in order**, but do *aptitude + coding + one CS subject every single day* in parallel. The week-by-week plan in [`curriculum/08-schedule/`](curriculum/08-schedule/) tells you exactly what to touch on each day.

```
curriculum/
├── 00-roadmap/          ← Read this first. The big picture and rules of the game.
├── 01-aptitude/         ← Quant, Logical Reasoning, Verbal/English.
├── 02-programming/      ← Python-first programming fundamentals (with C/Java notes).
├── 03-dsa/              ← Data Structures and Algorithms, topic-by-topic with problems.
├── 04-cs-core/          ← DBMS, OS, Computer Networks, OOPs.
├── 05-bonus-skills/     ← SQL, Git/GitHub, Cloud basics, Web, AI/ML literacy.
├── 06-resume-projects/  ← Resume template, project ideas, GitHub hygiene.
├── 07-interviews/       ← Company-wise patterns, HR, mock interview drills.
├── 08-schedule/         ← 24-week day-by-day calendar + daily/weekly templates.
└── 09-resources/        ← Curated free links, books, channels, communities.
```

## Daily rhythm (the single most important habit)

Pick **4 focused hours/day on weekdays**, **6-7 hours/day on weekends**. A typical weekday:

| Slot         | Duration | Activity                                              |
| ------------ | -------- | ----------------------------------------------------- |
| Morning      | 45 min   | Aptitude (1 sub-topic theory + 15 questions)          |
| Late morning | 75 min   | Coding / DSA (1 concept + 2-3 problems)               |
| Afternoon    | 45 min   | One CS core subject (read + summarise in own words)   |
| Evening      | 45 min   | Bonus skill OR project work OR English/communication  |
| Night        | 15 min   | Revision: redo 5 aptitude Qs + 1 DSA problem from memory |

Track this in a simple spreadsheet — consistency beats intensity.
