# 06 — Resume, Projects & Portfolio

A great resume + 2 polished projects + a clean GitHub is the **single biggest differentiator** between two students with similar test scores. Treat this folder seriously.

## Files

- [`resume-guide.md`](resume-guide.md) — 1-page resume template, what every section should say, dos & don'ts.
- [`project-ideas.md`](project-ideas.md) — 12 realistic project ideas mapped to the skill modules.
- [`linkedin-profile.md`](linkedin-profile.md) — LinkedIn checklist + how to actually use it.

## The 3-asset rule

Before any test, make sure you have:

1. **One-page resume PDF** — final, proofread, named `Firstname_Lastname_Resume.pdf`.
2. **GitHub profile** — 5+ repos with READMEs, profile README, photo.
3. **LinkedIn profile** — photo, headline, about, education, skills, your projects linked.

If any one of these is missing, fix it *before* solving another DSA problem.
