# The Placement Landscape (for service-based MNCs)

Before you study a single topic, understand *the game* you are preparing for. The format, syllabus, and difficulty for service-based MNCs is **very predictable** — that's the good news.

## Target companies and their assessment patterns

> Patterns change slightly every year, but the **shape** has been stable. Always re-check the official syllabus on the company portal a week before your test.

### 1. TCS — National Qualifier Test (TCS NQT) → TCS Ninja / Digital

- **Rounds:** Online test → Technical interview → Managerial → HR.
- **Online test sections:**
  - Numerical Ability (Quant) — ~20 Qs
  - Verbal Ability — ~20 Qs
  - Reasoning Ability — ~20 Qs
  - Programming Logic (MCQ on C/output/pseudocode) — ~10 Qs
  - Hands-on Coding — **2 problems**, ~45-60 min (1 easy, 1 medium)
- **Difficulty:** Easy to easy-medium. Speed + accuracy matter more than depth.
- **Languages allowed for coding:** C, C++, Java, Python, Perl.
- **TCS Digital** is the higher-package variant (₹7-11 LPA range). Same test, harder cutoff + advanced coding round.

### 2. Infosys — SP (System Engineer / System Engineer Specialist) and Power Programmer / DSE

- **Rounds:** Online test → Technical + HR (often combined for SP).
- **Online test sections:**
  - Mathematical Ability
  - Logical Reasoning
  - Verbal Ability
  - Pseudocode (MCQ)
  - Puzzles (data sufficiency / arrangements)
  - Hands-on Coding — 2 problems
- **Power Programmer / Specialist Programmer / Digital Specialist Engineer** (₹6-9 LPA+) has a tougher advanced coding round (1-2 hard problems, DSA-focused).

### 3. Tech Mahindra

- **Rounds:** Online test (incl. Versant English) → Technical → HR.
- **Sections:**
  - Aptitude (Quant + Reasoning)
  - English / Versant (spoken English scoring system)
  - Technical MCQs (C, DBMS, OS, Networks basics)
  - Essay writing (40-50 lines, paragraph on a generic topic)
  - Coding (1-2 problems)
- The **Versant English** test is a big differentiator — practice speaking out loud daily.

### 4. Other "must-prepare" mass recruiters (same prep covers all)

| Company           | Test highlights                                                          | Typical CTC        |
| ----------------- | ------------------------------------------------------------------------ | ------------------ |
| Wipro Elite NLTH  | Apt + Verbal + Logical + Written Communication (essay) + 2 coding        | 3.5-6.5 LPA        |
| Cognizant GenC    | Apt + Logical + Verbal + Coding (2 Qs)                                   | 4-9 LPA            |
| Accenture         | Cognitive + Technical + Coding (2 Qs) + Communication                    | 4.5-6.5 LPA        |
| Capgemini         | Pseudocode + Apt + English + Essay + Game-based + Coding                 | 4.25 LPA           |
| LTIMindtree       | Apt + Verbal + Logical + Coding                                          | 4-6.5 LPA          |
| HCL TechBee/Bharat| Slightly easier; aptitude + basic coding                                 | 1.7-4 LPA (varies) |
| Mphasis, Hexaware | Standard apt + coding pattern                                            | 3.5-5 LPA          |

> **Big insight:** if you prepare well for TCS NQT and Infosys, you are simultaneously ~80% ready for *every* other service company. So treat **TCS NQT + Infosys + Tech M** as your three benchmark tests.

## Eligibility (typical, verify each year)

- 60% / 6.0 CGPA throughout (10th, 12th, B.Tech) — most common.
- No active backlogs at the time of joining.
- Gap of ≤ 1 year (varies).
- For 3rd-year students: register for **TCS NQT in your final year's July-September** window and Infosys/Wipro/Cognizant **drives in August-November**.

**Aim for the higher track from Day 1.** The extra prep needed is *small* compared to the salary difference over 5 years.

## What this means for your prep

1. **Aptitude is non-negotiable.** All these tests have it. It is also the *easiest* free marks.
2. **Coding gets you the higher package.** Two clean coding solutions during the test = doubled CTC offer in many cases.
3. **Core CS subjects + projects** decide your interview result.
4. **English fluency** can quietly fail you in HR/Versant. Practice speaking, not just reading.
